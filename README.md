# Web Based Class Scheduling System

1. Description
The Web-Based Class Scheduling System is a web application developed as a graduation project to address the challenges of manual class scheduling 
at Debre Markos University (DMU) Bure Campus. The system provides an efficient and centralized platform for creating, updating, and managing class 
schedules. It supports role-based access for administrators and department heads, automatically checks for scheduling conflicts, manages academic resources, 
and allows students to view class schedules online. By automating scheduling tasks, the system improves accuracy, reduces administrative workload, and enhances 
the overall efficiency of academic schedule management.

2. Technologies Used
- PHP
- HTML
- CSS
- JavaScript
- MySQL

3. Features
# Authentication
- Secure login for Admin and Department Head using unique usernames and passwords.
- Role-based access control to protect system resources.

# Schedule Management
- Create, update, and delete class schedules.
- Detect scheduling conflicts before saving schedules.
- Modify schedules for cancellations or rescheduling.
- Assign instructors to courses based on their availability.

# Department Management
- Add, update, and remove department information.
- Manage department codes and department heads.
- Assign courses to batches for specific academic periods.

# Course Management
- Add, update, and delete courses.
- Manage course information, including course code, title, and credit hours.

# User Management
- Create, update, and deactivate user accounts.
- Manage user roles and permissions.

# Academic Period Management
- Add and update academic years and semesters.                                               
- Configure academic dates for scheduling.

# Public Schedule Access
- Allow students and visitors to view class schedules without logging in.

# Feedback Management
- Submit feedback about schedules or the system.
- Admins can review and mark feedback as read.

# Building and Room Management
- Register buildings, floors, and classrooms.
- Maintain classroom information for scheduling.
#  Security
- Secure authentication using encrypted passwords.
- Session management and input validation.
- Role-based authorization.

# System Quality
- User-friendly interface with simple navigation.
- Fast schedule generation and conflict checking.
- Reliable database management with data integrity.
- Designed for easy maintenance and future expansion.
